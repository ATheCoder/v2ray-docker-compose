راهنمای راه‌اندازی

Outline (گزینه ساده)

برای به‌کارگیری Outline نخست آنرا با یکی از گزینه های زیر دریافت کنید.

Android iOS Windows macOS

سپس برنامه Outline را باز کنید. بر روی + یا Add server بزنید. نوشته زیر را در جایگاه access key بگذارید.

بر روی دکمه ADD SERVER که پس از زمان کوتاهی پدیدار می‌شود بزنید. اکنون می‌توانید Proxy افزوده شده با نام name را به کار بگیرید.

Shadowsocks

Android iOS (ShadowLink) Windows macOS

همان access key بالا را می‌توانید برای Shadowsocks نیز به کار ببرید.

VMESS

Android (v2rayNG) iOS (ShadowLink) Windows (qv2ray) macOS (V2RayX)
